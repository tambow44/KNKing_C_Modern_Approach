### Project 2.1
Write a program that uses `printf` to display the following picture on the screen:
```
       *
      *
     *
*   *
 * *
  *
```
### Solution

#### 2.1.c
```c
#include <stdio.h>

int main(void)
{
	printf("       *\n");
	printf("      *\n");
	printf("     *\n");
	printf("*   *\n");
	printf(" * *\n");
	printf("  *\n");

   return 0;
}
```
